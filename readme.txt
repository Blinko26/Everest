R.I.A (Rush Into Air) : JetStream 
est un jeu de plateforme contre-la-montre où il faut sauver le père de la protagoniste nommée Aria 
: "Il y a 15.3 années, mon père fut emporté par les vents maléfiques. Mais aujourd'hui je vais le sauver grâce à mes pouvoirs du vent"


Ici il s'agira de terminer chacun des niveaux proposés le plus rapidement possible.

Pour lancer le jeu sous windows, executer le script riaW.bat
Pour lancer le jeu sous linux, executer le script ria.sh
Si ca ne fonctionne pas : "python menu.py" (ou "python3 menu.py")


Les touches utilisées dans les menus et dans le jeu sont :
(echap) Pour mettre pause ou revenir à la page précédente,

Les flèches directionneles :
(Up Arrow) Pour sauter et naviguer vers le haut dans les menus,
(Down Arrow) Pour naviguer vers le bas dans les menus,
(Left Arrow) Pour se deplacer horizontalement vers la gauche,
(Right Arrow) Pour se déplacer horizontalement vers la droite.

(Space) Pour sauter ou valider un choix dans les menus,

Et pour vous aider a terminer chaque niveaux le plus rapidement possible : 
(E) Un lancer de bulles d'air dans les niveaux,
(A) Un dash,


Un mode Demo est proposé dans celui-ci :
(E) active un bouclier empeche les degats
(Space) tire les bules d'air
